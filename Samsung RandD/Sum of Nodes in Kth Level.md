Given an integer ‘K’ and a tree in string format. print the sum of all elements at Kth level from root.

![image](https://s3.amazonaws.com/hr-assets/0/1573313905-8bbcb8a222-tree.png)

**Input Format**

integer k nexline string

**Constraints**

1 < k < 10

**Output Format**

single integer

**Sample Input 0**

```
2
(0(5(16()())(4()(9()())))(7(1()())(3()())))
```

**Sample Output 0**

```
24
```
